# AI Context Directory

This directory is the shared memory layer for all agents working in this repository.

## Purpose

- keep project context current
- avoid duplicate or conflicting work
- preserve architectural decisions
- make handoffs fast and reliable

## File Guide

- `PROJECT_STATE.md`: current project snapshot, scope, priorities, and constraints
- `DOC_ANALYSIS.md`: analysis of the current design and planning documents
- `DECISIONS.md`: canonical decisions and decision status
- `TASK_BOARD.md`: active claims, task status, and next steps
- `CHANGELOG.md`: time-ordered agent work log
- `HANDOFF_TEMPLATE.md`: standard handoff format

## Update Expectations

Every agent should treat this directory as part of the deliverable, not as optional notes.
